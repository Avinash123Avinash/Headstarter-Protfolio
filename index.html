Weather App

This project is a weather forecasting application that provides real-time weather information for any location. It includes features like current weather conditions, a 5-day weather forecast, and location-based weather updates. Users can search for weather details by city name and receive information such as temperature, humidity, wind speed, and weather descriptions. The app is designed with a responsive user interface using HTML, CSS, and JavaScript, ensuring a seamless experience across different devices.

Dummy data Generator

The Dummy Data Generator is a versatile tool designed to assist developers in creating realistic sample data for testing and development purposes. It offers a variety of data generation options, including names, addresses, emails, and more, to help users populate databases and applications with authentic-looking data. The tool supports customizable fields and formats, allowing for the generation of data that closely matches real-world scenarios. This makes it an invaluable resource for testing applications, ensuring that they function correctly with varied inputs and enhancing the overall development process..
